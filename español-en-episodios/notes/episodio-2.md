# Episodio 2

定冠词
- 阳性单复数
  - el / los
- 阴性单复数
  - la / las

名词
- 以下结尾的**通常**为阴性
  - a
  - dad
  - tad
  - ción
  - sión
- 以下结尾的**通常**为阳性
  - o
  - ema
- 其他的结尾
  - el autobús
  - el país
  - el reloj
  - el hombre
  - la calle
  - la paz
  - la razón
  - la piel
  - la mujer
- 单复数
  - 元音 `vocal + s`
  - 辅音 `consonante + es`
- 以 a 发音结尾的单词，单数定冠词只用 `el`
  - el agua, el alma

`IR` 的一般现在时变位
- 动词 `vivir` 的变位
  - Yo viv**o**
  - Tú viv**es**
  - Él / ella viv**e**
  - Nosotros / as viv**imos**
  - Vosotros / as viv**ís**
  - Ellos / as viv**en**
- 规则的动词还有
  - `recibir, escribir, abrir, subir`

`También` 和 `Tampoco`
- 肯定之后的肯定：`También`
- 否定之后的否定：`Tampoco`
- 肯定之后的否定：`No`
- 否定之后的肯定：`Sí`
- `gustar` 用 `A mí también / tampoco`
